Saving your workbook as a PDF

  1. Once the pages you want are unlocked, click Print / Save as PDF in the toolbar above.
  2. In the print dialog, set Destination to "Save as PDF" (Chrome, Edge, Firefox) or "Microsoft Print to PDF" (Windows) — on a Mac, use the PDF dropdown in the bottom-left of the print preview.
  3. Set Paper size to A4 and Margins to "None" or "Default."
  4. Turn on Background graphics (often under "More settings") so the seal stamps and shaded boxes print correctly.
  5. Click Save, choose where to keep the file — from there you can print physical copies from that PDF anytime.
